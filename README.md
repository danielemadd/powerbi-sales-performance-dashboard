## Project Overview
This Power BI sales analysis report focuses on analyzing sales performance data from 2014 to 2017.
The objective was to transform raw sales data into clear, actionable insights through structured reports and visual analysis.

## Tools Used
- Power BI
- Power Query

## Key Objectives
- Analyze revenue, profit, and order trends over time
- Evaluate Employees performance
- Define and track key performance indicators (KPIs)
- Evaluate product and category performance
- Support data-driven business decisions through reporting

## Data Preparation
- Cleaned and transformed raw sales data using Power Query
- Handled missing values and inconsistent data formats
- Ensured correct data types and relationships for accurate analysis

## Report Features
- High-level KPI summary
- Year-over-Year and time-based sales analysis
- Product and category performance visuals
- Employees performance visuals
- Interactive filters for focused analysis

## Key Insights
- Identified strong-performing products and categories over multiple years
- Identified Top employees and strong revenue areas
- Highlighted revenue trends and seasonal patterns
- Detected low-profit or declining sales segments

## Outcome
This project demonstrates the ability to build end-to-end Power BI reports, from raw data preparation to delivering business-ready insights.
